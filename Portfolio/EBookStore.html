動態網頁設計：模擬電子書商城

GitHub 原始碼

本作品為系外選修「動態網頁設計」課程的期末專案。核心目標是利用現代網頁開發技術，建構一個功能齊全的「模擬電子書商城」平台，實現從商品瀏覽到虛擬下單的完整電商體驗。

技術架構與功能亮點：

  • 開發環境： 使用 Angular 框架開發，並透過 ng-serve 進行前端高效渲染。
  • 模擬後端： 運用 json-server 快速搭建 mock RESTful API，模擬真實資料庫的資料存取行為。
  • 搜尋機制： 實作了靈活的搜尋引擎，支援透過「書名」或「作者」關鍵字精確篩選目標電子書。
  • 電商邏輯： 包含完整的商品明細檢視、購物車動態增刪，以及模擬下單結帳流程。
  • 版本控制： 專案全程使用 Git 進行版本管理，確保開發紀錄的嚴謹性。
