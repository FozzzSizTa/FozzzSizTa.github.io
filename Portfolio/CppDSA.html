C++ 資料結構與演算法 (2023-2024)

GitHub 原始碼

此專案集結了我在 2023 至 2024 年間，於大學課程中實作的各項 C++ 資料結構與演算法練習。透過親手編寫程式碼，我深入理解了這些核心計算機科學概念的運作原理，並鍛鍊了問題分析與解決能力。這不僅是學術上的探索，更是將理論知識轉化為實際應用能力的過程。

專案亮點與技術實作：

  • 排序演算法： 實作了多種經典排序演算法，如氣泡排序 (Bubble Sort)、選擇排序 (Selection Sort)、插入排序 (Insertion Sort)、快速排序 (Quick Sort) 和合併排序 (Merge Sort) 等。透過比較不同演算法的效率與適用場景，加深了對時間複雜度與空間複雜度的理解。
  • 河內塔 (Tower of Hanoi)： 運用遞迴 (Recursion) 思想解決經典的河內塔問題，清晰展示了遞迴函式的設計與執行流程，是理解遞迴邏輯的絕佳範例。
  • 老鼠走迷宮 (Maze Solver)： 實作了迷宮路徑搜尋演算法，通常採用深度優先搜尋 (DFS) 或廣度優先搜尋 (BFS) 等技術。這項實作不僅考驗了對圖形遍歷演算法的掌握，也展示了如何將抽象問題具體化為程式邏輯。
  • 其他資料結構： 還包含了鏈結串列 (Linked List)、堆疊 (Stack)、佇列 (Queue)、樹 (Tree) 等基本資料結構的實作，為更複雜的演算法奠定基礎。

這些專案不僅提升了我的 C++ 程式設計能力，更重要的是培養了我邏輯思維、演算法設計與除錯的關鍵技能。每個實作都是一次挑戰，也是一次成長，讓我對軟體開發的核心有了更深刻的體會。
