伺服器架設：多功能 Todo List (2024)

本作品為 2024 年「伺服器架設」課程的期末專案，是一套整合社群功能與權限管理的完整 Todo List 系統。 專案採用了 Ubuntu 作業系統搭配 MySQL 資料庫構建伺服器後端環境，並透過 HTML 與 CSS 打造具備響應式體驗的前端介面。

核心功能與特色：

  • 完善帳號系統： 支援使用者註冊與登入，確保個人資料的安全性與獨立性。
  • 科目任務管理： 使用者可以根據不同科目或類別新增與刪除 Todo List，提升任務管理的條理性。
  • 社群功能整合： 內建好友與群組系統。使用者可以添加好友、加入群組，並即時查看好友及群組成員的 Todo List 進度。
  • 細膩權限設定： 針對每個 Todo List 提供權限控制，可設定為「僅限自己可見」、「朋友可見」或「群組可見」，在分享與隱私之間達成平衡。

透過此專案，我深刻體會了從伺服器端環境建置到前端介面設計的完整流程，並在實作社群互動與權限邏輯的過程中，精進了資料庫結構設計與網頁應用開發的能力。
