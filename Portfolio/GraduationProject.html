Squeak Sage (2025)

GitHub (Code) Release Version

專案概述：模擬公司運營的創新遊戲體驗

「Squeak Sage」是我們五人團隊的畢業專題成果，旨在打造一個高度擬真的公司運營模擬遊戲。此專案不僅僅是一個遊戲，更是一個整合了前端、後端 API、資料庫、AI 模擬、客服系統、生成式 AI 地圖、版本檢測與更新機制，乃至於模擬儲值系統的綜合性平台。我們期望透過這款遊戲，全面模擬公司從提供服務到創造收入的各個環節，為玩家帶來前所未有的沉浸式體驗。

開發動機：AI Agent 啟發的遊戲創新

我們的靈感源於觀察 AI Agent 模仿人類遊玩《超級瑪利歐》的案例，這讓我們對 AI 在遊戲領域的潛力感到極度興奮。因此，我們決定開發一款類似《超級瑪利歐》的開源遊戲，並進一步利用 MLAgent 技術，訓練 AI 模仿玩家的行為模式，讓遊戲世界更具動態與挑戰性。這不僅是對技術的探索，更是對遊戲互動模式的一次大膽創新。

我的角色：Unity 前端開發與 API 整合核心

作為專案中 Unity 前端 的主要負責人，我獨立完成了所有遊戲物件的互動邏輯、UI 介面的設計與實作，以及與後端 API 的高效連結。這意味著從玩家的每一次點擊、每一次移動，到遊戲畫面的每一個元素，都凝聚了我在 Unity 引擎與 C# 程式語言上的深厚功力。確保前端的流暢性、響應性與視覺吸引力，是我的核心職責。

遊戲特色：豐富內容與商業模式的融合

在遊戲內容設計上，我們團隊集思廣益，創造了各式各樣的趣味怪物與獨特道具。玩家可以策略性地運用這些道具來對抗怪物，從而獲得更豐富、更多元的遊戲體驗。為了模擬真實的商業運營，我們甚至設計了引導玩家進行模擬儲值以購買道具的機制，藉此觀察與分析虛擬公司的收益模式。

此外，遊戲提供了「一般模式」與「無盡模式」，滿足不同玩家的挑戰偏好。我們也加入了「記分板」功能，鼓勵玩家之間相互競技，力求高分，增強了遊戲的社群互動性。在視覺呈現方面，我們設計了四種截然不同的地圖美術風格，大幅提升了遊戲的自訂性與重玩價值。值得一提的是，部分地圖的生成更運用了 生成式 AI 技術，展現了前沿技術在遊戲內容創作上的應用。

挑戰與成長：API 連結的磨礪與團隊協作

在整個專案的開發過程中，對我而言，最具挑戰性但也收穫最大的莫過於與後端 API 的連結。由於專案規模龐大，涉及多個模組與系統的協同運作，前端與後端的資料交換與邏輯同步顯得尤為關鍵。慶幸的是，我們團隊在專案初期便進行了充分的討論，確立了詳細的 API 格式與規範。我們也善用 GitHub 進行嚴謹的版本控制與分支管理，並結合 Jira 等專案管理工具，嚴格按照進度表推進開發，有效管理了開發過程中的程式碼衝突與協作效率。

然而，在開發過程中，團隊成員豐富的想像力與創造力，也讓我們不斷地在原定計畫之外，加入了更多創新的內容。這雖然增加了與後端溝通的複雜性，以及資料獲取的難度，但也正是在解決這些「意料之外」問題的過程中，我學會了更深入的跨部門溝通技巧、更靈活的問題解決策略，以及如何在高壓下保持高效的開發節奏。這段經歷極大地提升了我的技術實力與團隊協作能力。

榮譽與展望：專案成果的肯定

我們的專案成果不僅獲得了校內的肯定，榮獲 專研特優獎第一名 的殊榮，更有幸受邀參與 2025 電機資訊學院第 14 屆聯合資訊展。這些榮譽是對我們團隊努力的最高肯定，也讓我對未來的軟體工程之路充滿信心。透過「Squeak Sage」專案，我不僅掌握了 Unity 前端開發的精髓，更學會了如何在複雜的團隊協作環境中，將創新理念轉化為實際可行的產品。
