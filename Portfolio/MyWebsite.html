個人網站：GitPages 作品集與履歷

GitHub 原始碼

本網站本身即是一個作品集專案，旨在整合我的個人履歷、技術作品集與聯絡方式，並透過 GitHub Pages 進行部署，實現高效且易於維護的線上展示平台。網站採用現代化的卡片式設計，確保在不同裝置上都能提供優質的響應式使用者體驗。

核心技術與開發流程：

  • 前端技術： 主要使用 HTML 和 CSS 進行頁面結構與樣式設計，力求簡潔、高效且視覺吸引力。
  • 版本控制： 全程採用 Git 進行版本控制，確保開發過程的追溯性與協作效率。
  • 部署方式： 透過 GitHub Pages 服務，將靜態網站快速部署上線，實現零成本託管與便捷更新。

與 AI 協作的開發模式： 此專案的開發過程，充分展現了我與先進 AI 工具協同工作的能力。我主要利用了 GitHub Copilot 和 Gemini Code Assist 作為我的智慧助手，共同完成了網站的建構：

  • 精準的 Prompt Engineering： 我負責撰寫清晰、精確的指令（Prompt），引導 AI 生成符合需求的程式碼片段、CSS 樣式規則，以及頁面內容結構，大幅提升了開發效率。
  • 高效的程式碼審查與修正： 我對 AI 生成的程式碼進行嚴格的審查，不僅能識別潛在的錯誤或不符合最佳實踐的寫法，更能根據專案需求進行精準的修正與優化，確保程式碼的品質與穩定性。
  • 深入理解程式邏輯： 在與 AI 協作的過程中，我始終保持對程式碼邏輯的深入理解。這使我能夠在 AI 生成內容出錯時，迅速判斷問題所在，並提供更精確的修正方向或更有效的 Prompt，而非盲目接受。
  • 展現人機協作潛力： 本專案證明了我具備與 AI 工具高效協作的能力，能夠將其視為強大的生產力工具，而非替代品。這不僅加速了開發進程，也讓我能更專注於高層次的設計與問題解決。

這個個人網站不僅是我的作品集，更是我作為現代開發者，能夠駕馭新興技術、與 AI 協同創新，並對程式碼保持高度掌控力的最佳證明。
